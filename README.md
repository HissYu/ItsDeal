# ItsDeal
~~First secondary transaction website in China is online now!~~

本着服务大众（~~装逼~~）的目的，有志创建了此项目

基于Webpack4 & ThinkPHP 

根目录下为本项目的数据库导出文件

在前端根目录下运行`npm install` 即可完成依赖模块的安装~~也许~~

后端目录需要自行添加ThinkPHP框架一些已被忽略的内容：
> - thinkphp 文件夹及其所有内容
> - .travis.yml
> - composer.lock
> - application/.htaccess
> - application/database.php
> - vendor 文件夹及其所有内容
